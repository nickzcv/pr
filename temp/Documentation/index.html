Installation

Transfers is a HTML template so there's no real "installation" required, just unzip the main file you've downloaded from themeforest and after unziping you have your all template files ready to be used.


Folder structure

  • css folder: contains css stylesheet files. In these files it is defined how to display HTML elements.
  • fonts folder: contains all necessary font files for the website, including a preview of all custom icons with their respective css classes.
  • images folder: contains all necessary images for the website.
  • js folder: contains javascript files. These files provide enhanced user experience and dynamic effects.

CSS

There are three main CSS files in this template. Animate.css is for animation purposes and has no other effects on the website. Icons.css is used for custom font icons. We will take a look at the style.css file here. The first few lines are generic reset file. Many browser interpret the default behavior of html elements differently. By using a general reset CSS file, we can work round this. These lines also contains some general styling, such as anchor tag colors, font-sizes, etc. Keep in mind, that these values might be overridden somewhere else in the file.

Typography

Typography definitions can be found within /*typography*/ section.

Headings

HTML Headings are used to denote a title, emphasize a topic or give a summary to the main content. You can use headers in any context or area within the template. To create headings use the following html code:

<h1>This is a H1 Header</h1>

List Styles

To create a default unordered list with bullet, use the following Html code:

<ul>
<li>List item1</li>
<li>List item2</li>
<li>List item3</li>
</ul>

To create a default ordered list with numbers, use the following Html code:

<ol>
<li>List item1</li>
<li>List item2</li>
<li>List item3</li>
</ol>

Buttons

<a href="someLink.html" class="btn">This is a button</a>

Buttons can have additional (nested) classes such as:

white - makes the button background white
black - makes the button background black
grey - makes the button background grey
color - sets the button color depending on the color template used
small - small button
medium - medium button
large - large button
huge - huge button
full - makes the button full-width depending on the parent div size

Changing the color schemes

There are several ready to use color schemes provided. If you want to use one of them, all you need to do is open index.html or any other page and change the path to the desired css file in head section:
